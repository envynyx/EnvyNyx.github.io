What Is ASGard?



Simple

Based on an intuitive command structure, ASGard allows you to obtain and excute code faster than ever, update automatically, and do more with less code.

Secure

Asgard combines the security of modern asymmetric cryptography with the resilience and innovation of distributed ledgers to secure your source code acquisition process.

Scalable

We publish our signed package updates in a distributed ledger, similar to the one used in Bitcoin. Anyone with the Asgard client can run their own notary server for public or private use.

Simple

Most users will only ever use these three commands:

  • asgard license add (yourlicensekeyhere)
  • asgard get (packagename)
  • asgard update

Advanced users (DevOps managers, senior developers, security engineers, and system administrators) also benefit from the full capabilities we offer.

Secure

Asgard combines the security of modern asymmetric cryptography with the resilience and innovation of distributed ledgers to secure your source code acquisition process.

All packages are audited by security experts, cryptographically signed by Paragon Initiative Enterprises, and verified by independent notaries through a blockchain protocol.

For our cryptographic algorithms, we use libsodium, a portable version of NaCl—the world's most secure elliptic curve and high-speed cryptography library.

Scalable

We publish our signed package updates in a distributed ledger, similar to the one used in Bitcoin. Anyone with the Asgard client can run their own notary server for public or private use with these three commands:

  • asgard notary up starts the notary server (and generates a tag to share with clients the first time it's run).
  • asgard peer add (tag) adds a server to your list of trusted notaries.
  • asgard peer flush-untrusted removes the pre-installed notaries that we provide for convenience.
